# Pocket Gita

A free, open-source mobile app for reading and reflecting on the Bhagavad Gita.

- 18 chapters, 701 verses
- Sanskrit, transliteration, and 7 translations (English & Hindi)
- Daily shlok with optional reminder notification
- Bookmarks and personal notes
- Light / Dark / System theme
- No ads. No account. Everything stored on-device.

## Stack

- [Expo](https://expo.dev) (SDK 54) + React Native
- Expo Router v4 (file-based navigation)
- SQLite via `expo-sqlite`
- EAS Build for production builds

## Content

Gita text and translations sourced from the [VedicScriptures Bhagavad Gita](https://github.com/vedicscriptures/bhagavad-gita) dataset (LGPL-3.0).
